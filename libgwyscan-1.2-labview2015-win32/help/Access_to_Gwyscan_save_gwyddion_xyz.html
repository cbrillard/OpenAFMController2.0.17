Access to Gwyscan save gwyddion xyz.vi

Access to gwyscann save gwyddion xyz

savesXYZ data to Gwyddion XYZ file. Saves all channels from zdata. xypos, the xy positions, stores as interleaved arrays (x1, y1, x2, y2.......xn, yn) of size 2*n; z data channels are stored sequentially ( e.g. za1, za2, ....zan, zb1, z22,....zbn for channels za and zb).

error in (no error)

input parameters for save Gwyddion xyz

    		Z channel description

    an array of descriptions of the Z channels

      		description

      channel description for Gwyddion file

    		nchannels

    number of Z channels

    		zdata

    is measured values (height, signal error etx) stored sequentially (eg za1, za2..zan, zb1, zb2...zbn for channels za and zb)

      the measures Z values ( height , error signal etc) stored sequentially ( Za1, Za2,......Zan, Zb1, Zb2.....Zbn for channels Za and Zb)

    		xydata

    the interleaved x,y data available

    		xres

    is the x resolution of the data array to save; information for Gwyddion, may be 100

    		yres

    the y resolution of the data to save; information for Gwyddion, may be 100

    		filename

    filename of saved data
