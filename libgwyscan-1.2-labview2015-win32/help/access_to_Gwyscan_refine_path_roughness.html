Access to Gwyscan refine path roughness.vi

Creates new refined path based on difference of two previous scanning paths, representing coarse and fine scans

The function refines scan paths on basis of local data variance (roughness0 from previous scan, It splits the desired range (xreal, yreal, xoffset, yoffset) into xdiv X ydiv rectangles. It creates a preview regular array of previous scan data sets and it searches for rectangles where the local roughness is bigger than roughness. Tehn it creates a path going through all the found rectangles with pixel spacing of xstep and y step.

returns number of set elements or number of elements to allocate if xypos in NULL
