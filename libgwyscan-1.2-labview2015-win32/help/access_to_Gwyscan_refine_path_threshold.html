access to Gwyscan refine path threshold.vi

creates an new refined scan path based on the difference of two previous scanning paths, representing coarse and fine scans.

teh function refines the scan path on the basis of local changes detected in two previous scans. It splits the desired range (xreal, yreal, xoffset, yoffset) into xdiv X ydiv rectangles. It creates preview regular arrays of both the previous scan data sets provided and it searches for rectangles where the absolute local difference is bigger than the threshold. Then it creates a path going through all the found rectangles with pixel spacing of xstep and ystep.
