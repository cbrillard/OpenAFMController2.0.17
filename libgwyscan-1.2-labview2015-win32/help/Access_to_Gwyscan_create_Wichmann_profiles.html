Access to Gwyscan create Wichmann profiles.

Creates a Wichmann profile scans with progressive resolutions

The function creates a set of profiles formed by Wichmann sparse rulers. The scan positions are chosen such that all the possible point to point distances are in the range between 1 and the scan length (measured in multpiles of the smallest step) are present in the ruler at least once

Input parameters for Wichmann scan

    		yreal

    double

    		yoffset

    double

    		xreal

    double

    		xoffset

    double

    		nprofiles

    determines the number of profiles. Each profile is scanned continuously but the ordering of individual profiles provides progressive refinement of the image. In particular, the uppermost profiel is scanned first, then the lowermost profile, then the middle profile etc, recursively

    		n Wichmann

    determines the approximate smallest step in the profile , i.e. the ratio between xreal and the distance of two closest pointss on the profile. The total number of points on the path grows with the square root of this parameter. The function automatically determines the Wichmann ruler parameters r and s to minimise the number of positions for the given ruler length, but it does not guarentee the smallest step will be exactly as requested.
