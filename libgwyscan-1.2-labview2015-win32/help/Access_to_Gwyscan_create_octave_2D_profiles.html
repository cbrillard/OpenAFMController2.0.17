Access to Gwyscan create octave 2D profiles.vi

provides access to Gwyscancreate path octave 2d

that creates a two dimensional octave profiel scan starting from the finest resolution.

The function creates a two dimensional ocatve scan with the finest uniformly spaced block in the centre, cussounded by increasingly coarse positions. Each coarse spacing is double the previous one. The path form a spiral, improving isotropy and minimising the time needed to traverse the entire path. You can use this function to generate a plain square spiral path without any coarsening/refinement.

Parameter order determines the size of the basic block. At each power of 2 scale, the position set contains a continuous block of (2^order)+1 times (2^order+1) equidistant positions (as a subset; there may be more positions between them). The minimum order is 2 corresponding to a 5 x 5 basic block.

the smallest horizonatl step is equal to xreal/2^(order + noctaves) and the smalles vertical step is equal to

yreal/2^(order+noctaves)
