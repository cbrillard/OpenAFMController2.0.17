Access to Gwyscan create Wichmann profiles.

Creates a Wichmann profile scans with progressive resolutions

The function creates a set of profiles formed by Wichmann sparse rulers. The scan positions are chosen such that all the possible point to point distances are in the range between 1 and the scan length (measured in multpiles of the smallest step) are present in the ruler at least once as the direction between two positions.

Parameter nprofiles determines the number of profiles. Each profile is scanned continuously but the ordering of individual profiles provides progressive refinement of the image. In particular, the uppermost profile is scanned first, then the lowermost profile , then the middle profile etc recursively.

parameter n determines the approximate smalles step in the profile, ie. the ation between xreal and the distance of two closest points on the profile. Teh total number of points on teh path grws with the square root of this parameter. The function automatically determines the Wichmann ruler parameters r and s to minimise the number of positions for a given ruler length , but does not guarentee the smalles step will be exactly as requested.
