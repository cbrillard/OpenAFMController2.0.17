Access to Gwyscan create path space filling.vi

creates a scan path for equidistant measurments on a space -filling curve

Function creates a scan path by an order n-Hilbert space-filling curve. The positions lie in a regular grid with 2^points along eaxh axis. However, unlike usual regular sampling, ther is no disticntion between fast and slow axes. The sampling pattern in both directions is very similar.

The path starts in the top left corner (point with smalles x and y) and end in the top right corner (point with largest x and smallest y).

returns number of set elements.
