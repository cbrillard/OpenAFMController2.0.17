Access to Gwyscan create spiral path.vi

Creates a spiral path with progressive resolution inscribed on a rectangle

Teh function creates an Archimedian spiral. The spiral is sampled uniformly with a step that is chosen automatically to obtain the approximately equal resolution in the radial and tangential directions. The path always has exactly the requested number of points and it is incribed to the rectangle, that should prefereably be a square
