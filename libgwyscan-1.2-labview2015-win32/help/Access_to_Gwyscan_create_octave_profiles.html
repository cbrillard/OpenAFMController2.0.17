Access to Gwyscan create octave profiles.vi

Provides access to Gywscann_create_path_octave_profiles

The function creates a set of horizontal profiles, each consisting of a randomised actave sacan. A randomised octave profile contains, as subsets, uniform sequences of positions spaced at different powers of 2 of a base step.

nprofiles determines the number of profiles. Each profile is scanned contiuously but the ordereing of individual profiles provides progressive refinement of the image. in particular, the uppermost profile is scanned first, then the lowermost profile , then the middle profiel etc resursively.

Parameter order determines the size of the basic block. at each power of two scale, it is guarenteed that each profile contains a continuous block of (2^order)+1 equidistant positions (as a subset; there may be more positions between them)

parameter noctaves determines how many times the refinement progresses.

note that thes examples always start with the finest block. However, the positions of the finer scale blocks within the entire profile are randomised and the refinement can occur anywhere inside the profile

The smallest horizontal step is equal to xreal/2^(order + noctaves)
